# 🍕 Proyecto Web: Pizzería Home Delivery

Sitio web profesional desarrollado con **WordPress**, **Elementor** y **Astra**. Este proyecto demuestra mi capacidad para crear interfaces interactivas, funcionales y orientadas a la conversión de ventas.

## 🛠️ Detalles Técnicos
* **Navegación:** Header dinámico con acceso directo a pedidos vía WhatsApp.
* **UX/UI:** Categorización lógica de productos para facilitar la experiencia del usuario.
* **Interactividad:** Carruseles dinámicos y efectos hover para visualización de ingredientes.
